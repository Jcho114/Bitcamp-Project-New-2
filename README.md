# Real Estate Price Predictor - Bitcamp 2023

## Overview
This full-stack SERN application visualizes predictions of future median real estate prices in 6000+ zip codes in California.

## Tech Stack

### Full Stack
- MySQL
- Express
- React
- Node

### Machine Learning
- PyTorch

### Other Resources
- React Leaflet
